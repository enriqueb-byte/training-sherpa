## Quick workflow

1. **Kickoff:** Discovery + note persona + workflow bucket → see [`routing.md`](playbook/routing.md).
2. **Pick one primary objective** for each 30-minute session; use evidence in [`how-to-use.md`](playbook/how-to-use.md) to decide when to compress or defer.
3. **After the call:** Update the SP’s journey record; link Help Center articles in follow-up email from the module card.

## Maintaining Help Center links

Replace every `TODO:` link in [`modules/00-index.md`](modules/00-index.md) and individual module files with the current article URL. Prefer one curated bundle per module over duplicating steps here.
