# M04: Dedicated Phone Number

**Module ID:** `M04`  
**Typical phase:** Foundation (optional branch)  
**Default prerequisites:** M03 addressed; SP interested in tracked calls

### Learning outcome

The SP knows **what the dedicated number does**, **what they need** (e.g., privacy policy / terms if required), and has a **clear path** to complete or defer.

### Focus areas (30 minutes)

- Value: **call tracking**, **missed-call handling**, tie-in to **requests/leads** (use current product language).  
- **Requirements checklist:** what Jobber/your org requires for compliance.  
- If **privacy policy / terms** are the blocker: outline **fast path** (e.g., template + review) vs. **defer** with date.  
- Do **not** spend the whole session writing legal text, **escalate** to resources.

### Common failure modes

- Shame spiral on **legal docs**: normalize; reduce scope.  
- Perfectionism blocking **M05** workflow progress.

### Skip if …

- SP **does not want** a tracked number yet, note “deferred by choice.”

### Compress if …

- Number already live, quick review of **routing and reporting**.

### Defer if …

- **Policy block** and workflow not stable, **M05** first; return here when ready.

### Done looks like (observable)

- Number **active** OR **explicit deferral** with reason + follow-up date on journey record.
